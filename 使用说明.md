# TaskMate AI 科研助手 - 使用说明

## 一、应用简介

TaskMate AI 科研助手是一款集成了论文管理和 AI 对话功能的 Web 应用，帮助研究人员高效管理学术论文并与 AI 进行学术交流。

**访问地址**: http://localhost:3001

---

## 二、快速开始

### 启动应用

双击运行目录下的 **`启动 TaskMate.bat`** 脚本：
- 脚本会自动打开浏览器访问应用
- 同时启动后端服务

### 界面概览

```
┌─────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────┤

│                                                     │

├─────────────────────────────────────────────────────┤
│  [输入框...]                        [发送] [⏹]      │
```

## 三、核心功能

### 1. AI 对话

#### 发送消息
- 在底部输入框中输入问题或指令
- 点击「发送」按钮或按 `Enter` 键发送
- 支持流式输出，AI 逐步显示回答

#### 停止生成
- AI 回复过程中，点击「⏹」按钮可立即停止生成

#### 新对话
- 点击「**新对话**」按钮清空当前对话历史
- 开始全新的对话上下文

#### 思考过程
- 点击 AI 回复中的「💭 思考中」区域
- 可展开查看 AI 的推理过程

---

### 2. 文献上传

点击顶部「**📁 上传PDF**」按钮：
- 选择本地 PDF 文件
- 自动保存到本地文献库
- 支持从 arXiv ID 自动识别论文

#### 支持的命名方式
- `2406.03708.pdf` - 直接使用 arXiv ID
- `Attention Is All You Need.pdf` - 使用论文标题

---

### 3. 文献管理

点击顶部「**📚 文献管理**」按钮，打开文献管理面板：

#### 3.1 查看文献列表
- 显示所有已上传的论文
- 每条显示：论文标题、arXiv ID
- 支持搜索过滤

#### 3.2 搜索文献
- 在搜索框中输入关键词
- 实时过滤匹配的论文

#### 3.3 快捷操作
每个文献条目右侧有两个快捷按钮：
- 📁 - 在文件资源管理器中打开所在文件夹
- 🤖 - AI 自动重命名论文

#### 3.4 右键菜单
在文献条目上**右键点击**，弹出操作菜单：

| 功能 | 说明 |
|------|------|
| 📁 在文件资源管理器中打开 | 打开文件所在文件夹 |
| 🤖 AI重命名 | 使用 AI 自动提取论文标题并重命名文件 |
| 🗑️ 删除 | 删除论文文件（不可恢复） |

#### 3.5 AI 自动重命名
点击后系统会：
1. 解析 PDF 文件内容
2. 使用 DeepSeek API 提取论文标题
3. 自动重命名文件为论文标题

## 四、数据存储

### 本地文件位置

所有数据存储在 `data/` 目录下：

| 文件/文件夹 | 说明 |
|------------|------|
| `papers/` | PDF 论文文件存储目录 |
| `papers.json` | 已保存的论文元数据 |
| `tasks.json` | 任务管理数据 |
| `keywords.json` | 关键词收藏 |
| `journals.json` | 期刊收藏 |
| `research/` | 研究相关数据 |

## 六、注意事项

1. **首次使用**：确保 `启动 TaskMate.bat` 所在目录包含 `node_modules`
2. **文献上传**：PDF 文件大小建议不超过 50MB
3. **AI 重命名**：需要联网使用 DeepSeek API
4. **数据备份**：定期备份 `data/` 目录下的文件

---

## 七、故障排查

### 服务无法启动
- 检查 Node.js 是否已安装：`node --version`
- 确认端口 3001 未被占用

### 文献上传失败
- 检查 PDF 文件格式是否正确
- 确认 `data/papers/` 目录有写入权限

### AI 重命名失败
- 检查网络连接
- 确认 DeepSeek API Key 有效

---

## 八、技术栈

- **前端**: HTML + CSS + JavaScript
- **后端**: Node.js + Express
- **PDF 解析**: pdf2json
- **AI 服务**: DeepSeek API
